# File Server JS
```
--must be in /Users/kojiotsuka/workspace/file-server-js in terminal to 
--use this command.
--Also cntrl C to interrupt process and get pwd prompt
```

**Build project**
```
./node_modules/.bin/tsc --w
```

**Run server locally**
```
npm start
```

**Test in web browser**
```
open http://localhost:3000/
````

**Deploy to Heroku**
```
git push heroku master
```
